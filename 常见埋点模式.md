# 常见埋点模式库

## 模式 1: 列表项点击

### 场景
用户在列表中点击某个内容项（视频、新闻、评论等）

### 模板
```json
{
  "model": "[模块名]",
  "event": "点击[内容类型]",
  "type": "action",
  "params": ["from", "id", "position", "name"],
  "取值说明": {
    "from": "上级页面（例：主页推荐、新闻列表）",
    "id": "[内容类型]唯一标识",
    "position": "列表中的位置（从1开始）",
    "name": "[内容类型]名称"
  }
}
```

### 实例

#### 主页点击视频
```json
{
  "model": "主页",
  "event": "点击推荐视频",
  "type": "action",
  "params": ["from", "video_id", "position", "video_title"],
  "取值说明": {
    "from": "主页推荐区域",
    "video_id": "视频唯一标识",
    "position": "推荐列表中的位置",
    "video_title": "视频标题"
  }
}
```

#### 新闻列表点击新闻
```json
{
  "model": "新闻",
  "event": "点击新闻",
  "type": "action",
  "params": ["from", "news_id", "position", "news_title"],
  "取值说明": {
    "from": "新闻列表页",
    "news_id": "新闻唯一标识",
    "position": "列表中的位置",
    "news_title": "新闻标题"
  }
}
```

#### 搜索结果点击
```json
{
  "model": "搜索",
  "event": "点击搜索结果",
  "type": "action",
  "params": ["from", "keyword", "result_id", "position"],
  "取值说明": {
    "from": "搜索结果页",
    "keyword": "搜索关键词",
    "result_id": "结果唯一标识",
    "position": "结果排名"
  }
}
```

## 模式 4: 用户交互（赞/评论/分享）

### 场景
用户对内容进行赞、评论、分享等交互

### 模板
```json
{
  "model": "[模块名]",
  "event": "[操作名]",
  "type": "action",
  "params": ["from", "id", "action_type", "action_result"],
  "取值说明": {
    "from": "上级页面",
    "id": "目标对象ID",
    "action_type": "操作类型（1=赞, 2=评论, 3=分享）",
    "action_result": "操作结果（0=失败, 1=成功）"
  }
}
```

### 实例

#### 新闻点赞
```json
{
  "model": "新闻",
  "event": "点赞新闻",
  "type": "action",
  "params": ["from", "news_id", "action_result"],
  "取值说明": {
    "from": "新闻详情页",
    "news_id": "新闻唯一标识",
    "action_result": "0=取消赞, 1=点赞成功"
  }
}
```

#### 评论点赞
```json
{
  "model": "评论",
  "event": "点赞评论",
  "type": "action",
  "params": ["from", "comment_id", "post_id", "action_result"],
  "取值说明": {
    "from": "评论所在页面",
    "comment_id": "评论唯一标识",
    "post_id": "所属帖子ID",
    "action_result": "0=取消赞, 1=点赞成功"
  }
}
```

#### 分享内容
```json
{
  "model": "分享",
  "event": "分享内容",
  "type": "action",
  "params": ["from", "content_id", "content_type", "share_target"],
  "取值说明": {
    "from": "分享来源页面",
    "content_id": "内容唯一标识",
    "content_type": "内容类型（1=视频, 2=新闻, 3=评论）",
    "share_target": "分享目标（1=微信, 2=QQ, 3=微博）"
  }
}
```

## 模式 7: 应用生命周期

### 场景
应用启动、关闭、后台等生命周期事件

### 模板
```json
{
  "model": "用户行为",
  "event": "[事件名]",
  "type": "app_live",
  "params": ["from", "duration", "use_days"],
  "取值说明": {
    "from": "事件发生的页面",
    "duration": "时长（毫秒）",
    "use_days": "用户使用天数"
  }
}
```

### 实例

#### 启动APP
```json
{
  "model": "用户行为",
  "event": "启动APP",
  "type": "app_live",
  "params": ["from", "use_days", "ad_type"],
  "取值说明": {
    "from": "启动来源（1=直接启动, 2=推送启动, 3=分享启动）",
    "use_days": "用户本次安装后的累计使用天数",
    "ad_type": "广告未请求的理由"
  }
}
```

#### 离开APP
```json
{
  "model": "用户行为",
  "event": "离开APP",
  "type": "app_live",
  "params": ["from", "duration", "type", "free_memory", "used_memory"],
  "取值说明": {
    "from": "离开时所在页面",
    "duration": "打开返回至离开app的总时长（毫秒）",
    "type": "杀死进程/退出后台",
    "free_memory": "空闲内存（字节）",
    "used_memory": "已用内存（字节）"
  }
}
```

## 快速选择指南

| 场景 | 模式 | Event 类型 |
|------|------|-----------|
| 用户点击内容 | 模式1 | action |
| 内容展示 | 模式2 | exposure |
| 用户搜索 | 模式3 | action |
| 用户赞/评论/分享 | 模式4 | action |
| 页面进入/离开 | 模式5 | action |
| 视频播放 | 模式6 | action |

| 表单提交 | 模式8 | action |

| 错误异常 | 模式10 | action |

**使用提示**:
- 根据你的场景选择对应的模式
