# 埋点参数标准库

## 通用参数

### 来源追踪

| 参数 | 类型 | 说明 | 示例 |
|------|------|------|------|
| `from` | string | 上级页面/来源 | 主页、新闻列表、搜索结果 |
| `source` | string | 数据来源/渠道 | 推送、分享、直接进入 |
| `channel` | string | 频道/分类 | 推荐、热门、关注 |

### 内容标识

| 参数 | 类型 | 说明 | 示例 |
|------|------|------|------|
| `id` | string | 内容唯一标识 | 视频ID、新闻ID、用户ID |
| `name` | string | 内容名称 | 视频标题、新闻标题 |
| `type` | int/string | 内容类型 | 1=视频, 2=新闻, 3=评论 |
| `category` | string | 内容分类 | 足球、篮球、网球 |
| `tag` | string | 内容标签 | 热门、推荐、最新 |

### 位置信息

| 参数 | 类型 | 说明 | 示例 |
|------|------|------|------|
| `position` | int | 列表中的位置 | 1, 2, 3, ... (从1开始) |
| `index` | int | 数组索引 | 0, 1, 2, ... (从0开始) |
| `section` | int | 分区/分组 | 第1分区、第2分区 |
| `page` | int | 页码 | 1, 2, 3, ... |

### 时间信息

| 参数 | 类型 | 说明 | 示例 |
|------|------|------|------|
| `duration` | int | 时长（毫秒） | 5000, 10000 |
| `exposure_time` | int | 曝光时长（毫秒） | 3000 |
| `timestamp` | long | 时间戳 | 1609459200000 |
| `use_days` | int | 使用天数 | 1, 7, 30 |

### 用户信息

| 参数 | 类型 | 说明 | 示例 |
|------|------|------|------|
| `user_id` | string | 用户ID | 用户唯一标识 |
| `user_name` | string | 用户名 | 用户昵称 |
| `user_level` | int | 用户等级 | 1, 2, 3, ... |
| `is_login` | int | 是否登录 | 0=未登录, 1=已登录 |

### 交互信息

| 参数 | 类型 | 说明 | 示例 |
|------|------|------|------|
| `action_type` | int | 操作类型 | 1=赞, 2=评论, 3=分享 |
| `action_result` | int | 操作结果 | 0=失败, 1=成功 |
| `action_duration` | int | 操作耗时（毫秒） | 500 |

### 搜索相关

| 参数 | 类型 | 说明 | 示例 |
|------|------|------|------|
| `keyword` | string | 搜索关键词 | 用户输入的搜索词 |
| `search_type` | int | 搜索类型 | 1=全局, 2=本模块 |
| `result_count` | int | 搜索结果数 | 10, 20, 50 |
| `search_time` | int | 搜索耗时（毫秒） | 500 |

### 系统信息

| 参数 | 类型 | 说明 | 示例 |
|------|------|------|------|
| `free_memory` | long | 空闲内存（字节） | 1024000000 |
| `used_memory` | long | 已用内存（字节） | 512000000 |
| `max_memory` | long | 最大内存（字节） | 2048000000 |
| `network_type` | int | 网络类型 | 1=WiFi, 2=4G, 3=5G |
| `device_type` | string | 设备类型 | Android, iOS |

## 参数取值规范

### 布尔值
- `0` = 否/失败/未登录
- `1` = 是/成功/已登录

### 枚举值
- 使用整数表示: `1=选项A, 2=选项B, 3=选项C`
- 在取值说明中列举所有可能值

### 时间值
- 时长: 毫秒 (ms)
- 时间戳: Unix 时间戳 (秒或毫秒)
- 日期: ISO 8601 格式 (YYYY-MM-DD)

### 内存值
- 单位: 字节 (bytes)
- 示例: 1024000000 (1GB)

## 参数命名规范

### 蛇形命名法
- 使用小写字母和下划线
- 示例: `video_id`, `user_name`, `search_keyword`

### 避免缩写
- ❌ 错误: `vid`, `uid`, `kw`
- ✅ 正确: `video_id`, `user_id`, `keyword`

### 避免歧义
- ❌ 错误: `id` (不清楚是什么的ID)
- ✅ 正确: `video_id`, `user_id`, `post_id`

## 常见参数错误

### ❌ 错误示例

```json
{
  "vid": "12345",           // 缩写，应该是 video_id
  "uid": "user_123",        // 缩写，应该是 user_id
  "kw": "梅西",             // 缩写，应该是 keyword
  "id": "12345",            // 不清楚是什么的ID
  "time": 5000,             // 不清楚是什么时间
  "pos": 3                  // 缩写，应该是 position
}
```

### ✅ 正确示例

```json
{
  "video_id": "12345",
  "user_id": "user_123",
  "keyword": "梅西",
  "post_id": "12345",
  "duration": 5000,
  "position": 3
}
```
